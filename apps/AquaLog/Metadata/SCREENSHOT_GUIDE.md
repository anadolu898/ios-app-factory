## How to Generate

1. Use ChatGPT-4o with image generation
2. Copy each prompt above
3. If the first generation isn't right, ask ChatGPT to adjust (move elements, fix text, change colors)
4. Download at 1320×2868 for iPhone 6.9"
5. For iPhone 6.3" (1206×2622), you can resize the same images
6. Save files as: `screenshot_1_dashboard.png`, `screenshot_2_timeline.png`, etc.
7. Put them in `apps/AquaLog/Metadata/screenshots/`

## Text Overlay Tips
- Keep overlay text SHORT (3-5 words max)
- Use white text on colored gradient banner
- Place at top 15% of image
- Font: SF Pro Display Bold or similar clean sans-serif
- The screenshot content should speak for itself — overlay is just a headline

## What Apple Reviewers Look For
- Screenshots should show real app functionality, not just marketing graphics
- At least one screenshot must show the main feature
- Include variety: dashboard, detail view, widget, settings
- Dark mode variant is optional but impressive
